Написать простое REST приложение - Библиотеку. Простой CRUD для нескольких таблиц: USERS, BOOKS (остальные таблицы, если понадобятся, на усмотрение пользователя).
Пользователь может брать книги во временное пользование, о чём должна быть соответствующая пометка с датой выдачи и датой возврата (требуемая и фактическая).
Также должна быть доступна следующая статистика:
1. Показать все книги (вместе с пользователями), которые, в данный момент, на руках.
2. Показать всех пользователей, которые брали книгу (включая того, у которого она в данный момент на руках) с датами взятия и возврата.

Обязательно использовать:
UI: REST + JSON + Spring MVC
BL: Spring IoC, Spring Data
DBL: MySQL
Server: Tomcat
Build tool: Maven

Решение представить в виде maven артефакта. Обязательно использование Spring Boot.
Приложение должно быть представлено вместе с исходниками, скриптами для базы данных и инструкциями по сборке и запуску.
